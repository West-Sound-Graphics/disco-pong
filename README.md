# Disco Pong

A vibrant, disco-themed Pong game built with Three.js. Play against the computer in this classic arcade game with a rainbow twist!

## Features

- **User vs Computer**: Challenge the AI opponent
- **Arrow Key Controls**: Use up and down arrows to control your paddle
- **Score Keeping**: First to 10 points wins
- **Disco Theme**: Rainbow background with geometric patterns and glowing elements
- **Dynamic Colors**: All game elements cycle through rainbow colors
- **Disco Music**: Built-in synthesized disco background music with play/pause controls
- **Sound Effects**: Distinct sounds for wall bounces and paddle hits
- **Custom Settings**: Title screen controls for ball speed (Slow/Mid/Fast) and AI skill (Easy/Normal/Hard)
- **3D Graphics**: Powered by Three.js for smooth gameplay

## How to Play

1. Open `index.html` in your web browser
2. Click the "🎵 Play Disco Music" button to start the background music
3. Use the up and down arrow keys to move your paddle
4. Hit the ball back to the computer
5. Score points by getting the ball past the computer's paddle
6. First to 10 points wins!

## Controls

- **Up Arrow**: Move paddle up
- **Down Arrow**: Move paddle down
- **Music Button**: Toggle disco background music on/off

## Audio Features

- **Background Music**: Synthesized disco beat and melody
- **Wall Bounce Sound**: High-pitched "ping" when ball hits top/bottom walls
- **Paddle Hit Sound**: Low "thud" when ball hits paddles

## Technologies Used

- Three.js for 3D graphics
- HTML5 and CSS3 for styling
- JavaScript for game logic

Enjoy the disco vibes while playing!
