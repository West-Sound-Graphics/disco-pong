# Disco Pong

A vibrant, disco-themed Pong game built with Three.js. Play against the computer in this classic arcade game with a rainbow twist!

## Features

- **User vs Computer**: Challenge the AI opponent
- **Arrow Key Controls**: Use up and down arrows to control your paddle
- **Score Keeping**: First to 10 points wins
- **Disco Theme**: Rainbow background with geometric patterns
- **3D Graphics**: Powered by Three.js for smooth gameplay

## How to Play

1. Open `index.html` in your web browser
2. Use the up and down arrow keys to move your paddle
3. Hit the ball back to the computer
4. Score points by getting the ball past the computer's paddle
5. First to 10 points wins!

## Controls

- **Up Arrow**: Move paddle up
- **Down Arrow**: Move paddle down

## Technologies Used

- Three.js for 3D graphics
- HTML5 and CSS3 for styling
- JavaScript for game logic

Enjoy the disco vibes while playing!
